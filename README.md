# UI Testing Playground - Cypress Tests

This repository contains simple Cypress tests for [UI Testing Playground](https://uitestingplayground.com/).  
Each test covers a different page and scenario to practice Cypress automation.

## Pages Tested

- Dynamic ID
- Class Attribute
- Hidden Layers
- Load Delay
- AJAX Data
- Client Delay
- Click
- Text Input
- Scrollbars
- Disabled Input
